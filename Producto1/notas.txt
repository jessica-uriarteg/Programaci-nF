ls. presenta la lista de un directorio. Ej. ls -al
cd. cambio de directorio. Ej. cd[location]
file. obtener informacion sobre que tipo de archivo es. Ej.  file nota.txt
man. busca la pagina del manual para encontrar informacion sobre un comando en especial. Ej. man cd
mkdir. crear un directorio. Ej. mkdir ~/progfortran
rmdir. borrar un directorio. Ej. rmdir progfortran
touch. crear un archivo en blanco. Ej. touch ejemplo1
cp. copiar un archivo o directorio. Ej. cp nota.txt
mv. mover un archivo. Ej. mv nota.txt progfortran
rm. borrar un archivo. Ej. rm ejemplo1
vi. editar un archivo. Esc para salir de vi. Ej. vi nota.txt
cat. juntar dos archivos o ver un archivo. Ej. cat progfortran
ls -l. ver los perimsos de un archivo. Ej. ls -l /home/hjvalenzuela/nota.txt
chmod. cambiar los permisos de un archivo o directorio. 
sed. buscar datos y remplazarlos con otros. Ej. sed s/uno/dos/g nota.txt
uniq. borra lineas duplicadas en el archivo/documento. Ej. uniq nota.txt
tac. muestra la informacion de un archivo de abajo hacia arriba. Ej. tac nota.txt
egrep. ver lineas de datos que siguen un patron. Ej. egrep [aeiou] nota.txt
top. ver los programas que esta corriendo. Ej. top
nl. numera la lista de datos en tu archivo. Ej. nl nota.txt
